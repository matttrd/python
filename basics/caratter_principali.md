# Caretteri principali

| Character | Names                                 |
|-----------|---------------------------------------|
| `+`       | plus                                  |
| `-`       | minus, dash                           |
| `_`       | underscore                            |
| `*`       | star, asterisk                        |
| `/`       | forwardslash (it's leaning forward)   |
| `\`       | backslash (it's leaning back)         |
| `=`       | equals sign                           |
| `%`       | percent sign                          |
| `.`       | dot                                   |
| `,`       | comma                                 |
| `:`       | colon                                 |
| `?`       | question mark                         |
| `!`       | exclamation mark                      |
| `<` `>`   | less-than and greater-than signs      |
| `'` `"`   | single quote and double quote         |
| `#`       | hashtag                               |
| `()`      | parentheses                           |
| `[]`      | square brackets, brackets             |
| `{}`      | curly braces, braces, curly brackets  |
